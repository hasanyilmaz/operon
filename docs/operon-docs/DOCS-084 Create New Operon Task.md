# Create New Operon Task

**Create New Operon Task** opens the full [[DOCS-020 Task Creator|Task Creator]]. It is the safest starting point when you want Operon to guide you through making a task, rather than typing task syntax by hand.

## What it does

The command opens the Task Creator dialog, where you fill in a task through fields instead of writing them on a line. It does not depend on your cursor position, so you can create a task from anywhere in the vault.

## When to use it

- You do not want to write task syntax by hand.
- You want to create a task from anywhere, not only at the cursor.
- You want to choose fields such as status, priority, dates, parent task, estimate, recurrence, or links before the task exists.

For a lighter, in-place option that follows your cursor, use [[DOCS-085 Create or edit inline task|Create or edit inline task]] instead.
